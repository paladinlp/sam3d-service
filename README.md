# sam3d_service_repo

This repository contains the standalone `sam3d_service/` folder used to add a
web UI and API service layer on top of Meta's `sam-3d-objects` repository.

## Layout

- `sam3d_service/`: copy this folder into the root of an existing
  `sam-3d-objects` checkout.

## Usage

1. Copy `sam3d_service/` into the official `sam-3d-objects` repo root.
2. Follow `sam3d_service/README.md` and `sam3d_service/DEPLOY.md`.

This repo does not include the upstream `sam-3d-objects` source code or model
weights.
